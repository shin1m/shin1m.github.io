xemmai A dynamic programming language

xemmai is a dynamic programming language. It has lambda closures, classes, exceptions, modules, threads, and fibers. It aims to be a general purpose glue language.

It is implemented in C++ and has C++ friendly API to implement extension modules. Internally, codes are executed on a bytecode interpreter. It also has a concurrent garbage collector to utilize multiple processors.

It runs on linux, windows, and hopefully other unix systems.
